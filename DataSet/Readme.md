# Описание Dataset
### [Ссылка на Dataset](https://www.kaggle.com/sakhawat18/asteroid-dataset?select=dataset.csv) 

### Содержание
 Dataset содержит описание астрономических объектов таких как кометы, 
 астеройды, метеориты, сверхмалые планеты. 
### Dataset содержит 958525 строки и 20 столбцов. 
### В качестве параметра астрономичсекого объекта выступают следующие показатели
1. SPK-ID - SPK-ID номер объекта
2. Object ID - индетификатор внутренней базы данных объекта
3. Object fullname - полное обозначение объекта
4. pdes - первичное обозначение объекта
5. name - имя объекта
6. NEO - флаг объекта, сближающегося с Землей
7. PHA - флаг потенциально опасного объекта
8. H - параметр абсолютной величины
9. Diameter - диаметр
10. Albedo - альбедо
11. Diameter_sigma - погрешность в диаметре объекта
12. Orbit_id - ID орбиты
13. Epoch - эпоха осцеляции в модифицированной форме юлианского дня
14. Equinox - равноденствие системы отсчета
15. e - эксцентриситет
16. a - большая полуось
17. q - радиус в перигелии 
18. i - наклонение орбиты
19. tp - Время прохождения перигеля орбиты
20. moid_ld - Минимальное расстония до перечечения орбиты Земли

## Данная выборка может быть полезна в определение потенциально опасных астрономических объектов, как для Земли, так и для космических аппаратов находящихся на орбите Земли
